Theme hochladen und aktivieren

Entpacken Sie die geladene .zip Datei und laden den kompletten Theme-Ordner mit Namen "property" auf Ihren Webserver in das WordPress Theme Verzeichnis. Standardmäßig heißt dieses: wordpress/wp-content/themes. Anschließend kann das neue Theme im WP Admin-Bereich unter “Design – Themes” aktiviert werden. Alternativ können Sie die Theme .zip-Datei auch direkt unter "Design - Themes" hochladen und installieren.

Anschließend werden Sie aufgefordert, die empfohlenen Plugins zu installieren:

  • Query-Multiple-Taxonomies - für die Auswahl der Immobilien
  • Easy Google Fonts - um beliebige Google Webfonts auszuwählen
  • Contactform7 - um evtl. weitere Formulare einzurichten.
  • Post Type Switcher - um Immobilien nach Verkauf/Vermietung direkt in die Rubrik "Referenzen" zu verschieben

 

1. Seiten-Templates

Wenn eine neue Seite erstellt oder eine vorhandene editiert wird, können Sie in der Edit-Ansicht rechts unter “Attribute” das gewünschte Template auswählen. Nach dem Speichern wird das neue Template für die Seite genutzt. Folgende Templates stehen zur Verfügung:

  • Standardseiten mit Sidebar => Standardtemplate
  • Standardseiten mit voller Breite => Template: Seiten ohne Sidebar
  • Eine Liste aller Immobilien => Template: Immobilienliste
  • Eine Liste aller Services => Template: Serviceliste
  • Seite mit dem Kontaktformular => Template: Kontaktformular,
  • Startseite mit ausgewählten Immobilien, Infoboxen und Content => Template: Startseite1
  • Startseite mit Bildergalerie, InfoboxenContent und Immobilien-Karussel => Template: Startseite2
  • Startseite mit Bildergalerie, Content und Immobilien-Karussel => Template: Startseite3
  • Startseite mit individuell positionierten Elementen => Template: Individuelle Startseite
  • Seite mit Referenzobjekten mit Sidebar => Template: Referenzobjekte mit Sidebar
  • Seite mit Referenzobjekten ohne Sidebar => Template: Referenzobjekte ohne Sidebar

Wenn z.B. eine Startseite als Template ausgewählt wird, werden die entsprechenden Inhalte automatisch zusammengestellt. Bitte beachten Sie, dass zuerst Seiten unter der Rubrik "Immobilien" erstellt werden und evtl. Bilder in den Theme-Einstellungen hochgeladen werden müssen, damit diese automatisch auf den Startseiten angezeigt werden können.

2. Die Startseite

WordPress listet nach der Installation standardmäßig die aktuellsten Beiträge (Posts) auf der Startseite. Dies kann unter “Einstellungen / Lesen” verändert werden. Erstellen Sie dazu eine neue Seite und weisen ihr das gewünschte Startseiten-Template zu, s.o. Anschließend können Sie in der Liste unter “Einstellungen / Lesen” die gewünschte Seite auswählen und als Startseite aktivieren. Nun startet WordPress auch mit der ausgesuchten Seite.

3. Grundsätzliche Theme-Einstellungen

Unter dem Admin-Menupunkt "Design/Theme-Einstellungen finden Sie alle individuellen Einstellungsmöglichkeiten.

Basiseinstellungen

  • Logo Upload oder alternativ
  • Eingabe von Website Titel/Untertitel
  • Alternative Währung für die Immobilienseiten

Startseiten

  • Willkommenstexte bzw. Slogans
  • Einstellungen für den internen Slider
  • Upload für Bilder (Template: Startseite2 und 3)
  • Zusammenstellung der individuellen Startseite aus beliebigen Abschnitten

Infoboxen

  • Beliebig viele Infoboxen für die Starsteite inkl.
  • Icon-Feld, Headline-Feld, Textfeld, Buttonbeschriftung und Link

Layout

  • Hintergrundfarben und Bilder
  • Textfarbe, Headlinefarbe, Linkfarbe
  • Farbe des Menüs
  • Hintergrundfarben für Button und Icons
  • Hintergrundfarbe für die Sidebar-Headlines

Button und Kontakt

  • Headline für die Immobilien-Auswahlbox
  • Linkziel für die Immobiliensuche
  • Linkziel für den Kontakt
  • Empfängeradresse für die eMails
  • Text für die Betreffzeile der eMails
  • Text für Anzeige nach dem Versand der eMails
  • Einstellungen für die Widerrufsbelehrung

Social Media

  • Eingabe der Ziel-URLs zu den einzelnen Diensten (Facebook, Twitter, Xing usw.)

Blog

  • Überschrift über die Blogseite (index.php), wenn gewünscht
  • Auswahl der Darstellung

SEO

  • Allgemeine Meta-Tags, die standardmäßig genutzt werden können.
  • (Auf allen Seiten können title, description und keywords auch individuell überschrieben werden.)

3a. Einstellungen für die Widerrufsbelehrung

Wenn Sie Ihre Besucher bei Anfragen direkt über das Widerrufsrecht belehren möchten, dann können Sie entsprechende Voreinstellungen übernehmen. Wenn Sie nicht über das Widerufsrecht belehren oder keine automatische Antwortmail inkl. Hinweis zum sofortigem Tätigkeitsbeginn versenden möchten, können Sie die Auswahlboxen auf "Nein" setzen und die übrigen Felder einfach leer lassen.


Folgender Ablauf ist voreingestellt:

  • Bei einer Anfrage über den Button auf den Immobilienseiten wird zum Kontaktformular (Template Kontaktformular) weitergeleitet.
  • Dort werden Besucher aufgefordert, die verlinkte Widerrufsbelehrung vor Absenden des Formulars zu bestätigen.
  • Nach Absenden des Formulars werden zwei Mails verschickt:
  • 1. An den Besucher - inkl. Immobiliendaten und Hinweise zum Widerruf zwecks sofortigem Tätigkeitsbeginn.
  • 2. An die in WP hinterlegte Mail-Adresse (d.h. an Sie) - inkl. Kontaktdaten des Besuchers und Immobiliendaten.

Den Text zur Widerrufsbelehrung (Datei: widerrufsbelehrung.php) und den Inhalt der automatischen Mail (Datei: wrmail.php) können Sie, falls erforderlich, direkt in WordPress (Design/Editor) oder lokal mit einem Texteditor bearbeiten.
Sämtliche Textvorlagen entsprechen den Vorgaben lt. immobilienscout24.de. Eine Gewährleistung für Aktualität, Vollständigkeit und Richtigkeit kann selbstverständlich nicht gegeben werden!

4. Einstellungen auf den einzelnen Seiten

Auch auf den einzelnen Seiten (d.h. allen Immobilien/Services/Artikeln/Seiten) lassen sich zusätzliche Einstellungen vornehmen:

  • Anzeige dieser Seite auf der Startseite aktivieren (wenn Sie möchten, dass z.B. eine bestimmte Seite oder Serviceleistung auf der Startseite in einer Teaserbox gelistet wird)
  • Daten für die Immobilien (alle Datenfelder sind optional und werden nur angezeigt, wenn Daten enthalten sind)
  • Zusätzliche individuelle Datenfelder für Immobilien können erstellt werden
  • Unbegrenzter Upload von Bildern für die Galerie
  • Upload von Infos als pdf
  • Einbindung von Google Map Standorten*
  • Meta-Tags "title/description/keywords" für diese Seite

Den passenden Google-Code mit den Adressdaten des Objektes bekommen Sie auf maps.google.de. Geben Sie dort die (ungefähre) Adresse ein und klicken anschließend auf das Hyperlink-Symbol oben links. Dort wird Ihnen der entsprechende HTML-Code zur Verfügung gestellt. Kopieren Sie ihn und fügen ihn komplett in das dafür vorgesehene Feld in den Immobilien-Daten.

5. Blog / News

Nutzen Sie die Blogfunktion, z.B. um Neuigkeiten zu veröffentlichen, zu kategorisieren und zu archivieren. Die Kommentarfunktion kann nach Wunsch entweder komplett oder von Fall zu Fall auf den einzelnen Artikelseiten deaktiviert werden.

Um eine Übersichtseite der aktuellsten Artikel zu erhalten, erstellen Sie eine leere Seite mit einem Standardtemplate und benennen Sie sie nach Wunsch (z.B. Aktuelles). Im Admin-Menu: Einstellungen/Lesen können Sie nun Ihre eben erstellte Seite als "Artikelseite" eintragen. WordPress weist dieser Seite nun automatisch das richtige Template (index.php) zu, die Artikel werden entsprechend angzeigt.

Weitere Einstellungen, z.B. zur Anzeige von Datum, Autor, Kategorie usw. können unter "Design/Theme-Einstellungen" vorgenommen werden.

6. Haupt-Menu - individuelle Menus für die Sidebar

Im Admin-Bereich unter “Design/Menüs” werden die Menus aus den vorhandenen Seiten* zusammengestellt. Per Drag&Drop können die gewünschten Seiten zum jeweiligen Menu gefügt werden. Dort können Sie auch die Bezeichnungen für die Menu-Einträge ändern, z.B. lange Seitentitel einfach verkürzen.

* alle vorhandenen Datensätze sind als Menupunkt nutzbar, d.h. Seiten, Artikel, Angebote genauso wie Kategorien, Schlagworte oder Angebotskategorien. Auch externe URLs sind nutzbar. Um alle Menuoptionen zu sehen, klicken Sie oben auf "Optionen einblenden" und aktivieren die entsprechenden Checkboxen. WordPress zeigt standardmäßig nicht alle vorhandenen Optionen an.

Um das DropDown-Menu für die Menuleiste oben zu erhalten, erstellen Sie ein neues Menu und fügen dort die gewünschten Seiten ein. Um die Seiten hierarchisch anzuordnen, schieben Sie die entsprechenden Unterpunkte ein wenig nach rechts, bis diese automatisch ihre eingerückte Position finden. Auf diese Weise entsteht die gewünschte Menuhierarchie. Links in der obersten Box “Anordnung im Theme” wählen Sie dann das erstellte Menu für die Darstellung in der Menuleiste oben aus.

Bitte beachten Sie, dass eine Auswahl der Menufarbe (Theme-Einstellungen) und eine responsive Darstellung erst möglich ist, wenn ein hier erstelltes Menu der "Hauptnavigation im Header" zugeordnet wurde.

Ebenso können beliebig viele weitere Menus aus den vorhandenen Seiten erstellt werden. Diese Menus stehen Ihnen dann anschließend unter "Design/Widgets" zum Einbinden in die verschiedenen Sidebars zur Verfügung.

Automatisch hervorgehobene Menüpunkte

Wenn Sie für alle relevanten Seiten mit Immobilien (Archivlisten, Taxonomielisten) auch einen Hauptmenupunkt z.B. "Immobilien" (Template: Immobilienliste) hervorheben möchten, so fügen Sie bei diesem die CSS-Klasse "immo" hinzu.

Wenn für alle relevanten Seiten oder Listen mit News auch ein Hauptmenupunkt z.B. "News" hervorgehoben werden soll, dann fügen Sie hier die CSS-Klasse "blog" hinzu.

Wenn für alle relevanten Seiten oder Listen mit Referenzen auch ein Hauptmenupunkt z.B. "Referenzen" hervorgehoben werden soll, dann fügen Sie hier die CSS-Klasse "referenz" hinzu

Bitte beachten Sie, dass WordPress nicht standardmäßig alle Menuoptionen einbindet. Klicken Sie oben auf den Button "Optionen", um alle vorhandenen Rubriken und/oder Einstellungsmöglichkeiten anzuzeigen.

7. Sidebars und Widgets

Es stehen verschiedene Sidebars für verschiedene Templates zur Verfügung. So können z.B. auf Standardseiten andere Informationen eingebunden werden als auf Serviceseiten oder Newsartikeln. Um vorab erstellte Menus einzubinden, nutzen Sie das Widget "Individuelle Menus".

 

Für die Ausgabe der neuesten Immobilien stehen zwei zusätzliche Widgets zur Verfügung:

+ "Immobilien-Liste" und "Immobilien-Slider"

8. Muster-Content übernehmen

Wenn Sie eine ganz neue Webseite mit WordPress aufbauen, können Sie Musterinhalte importieren und diese überschreiben. Sie finden im Theme-Ordner eine xml-Datei mit dem Muster-Content der Demo-Seite, die Sie zu diesem Zweck in Ihre WordPress Installation importieren können. Gehen Sie dazu im Admin-Menu auf "Werkzeuge/Daten importieren" und folgen den Anweisungen.

Bitte beachten Sie, dass die Bilder, die auf der Demo-Seite eingesetzt wurden, nicht geladen oder genutzt werden dürfen.
